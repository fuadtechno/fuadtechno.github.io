API Documentation

Fuadsoft APIs enable researchers and developers to programmatically access PhD-core analytics and automated deployment workflows.

GET

Fetch Research Analytics

Retrieves a comprehensive data object containing project-specific performance metrics and system integrity reports.

POST

Trigger Deployment

Initiates an automated CI/CD pipeline for the specified research environment. Requires a valid PhD-Core Auth Token.
